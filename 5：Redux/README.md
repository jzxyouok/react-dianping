React-router基础知识，详情参考[这里](./docs/README.md)

本例子中用的"react-redux": "^4.4.5"

其中app/redux-demo.js文件是<https://github.com/reactjs/redux>中的入门demo，在app/index.js打开下面注释就可以运行

	// 引用并执行 redux-demo
	import fn from './redux-demo.js'
	fn()

运行

	npm install //安装依赖

	npm run start //启动项目

	npm run build //上线和优化项目
